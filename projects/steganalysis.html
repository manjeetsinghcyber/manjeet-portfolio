Steganography Analysis (Steghide)

Cybersecurity Project by Manjeet Singh

Project Summary

In this project, I analyzed an image to find hidden data using steganography tools. I used Steghide to extract secret information from a JPEG file. This helped me understand how attackers may hide data and how digital forensics teams investigate files.

Tools Used

  • Steghide
  • Linux Terminal (Kali Linux)
  • strings command

Steps Performed

1. Used strings to scan the image for clues:

strings GumpyCat.jpeg | less

2. Found the hidden password near the bottom of the output.

3. Extracted the hidden file using Steghide:

steghide extract -sf GumpyCat.jpeg

4. Entered the password to reveal the secret text file.

5. Opened the extracted file to view the hidden message:

cat secret.txt

What I Learned

I learned how steganography can hide data inside images and how forensic analysts recover hidden information. This project taught me how attackers may hide passwords or secret files inside media. I also learned to use Linux tools such as strings and steghide.
