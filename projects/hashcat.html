Hash Cracking with Hashcat

Cybersecurity Project by Manjeet Singh

Project Summary

In this project, I used Hashcat to crack a password hash. The goal was to understand how weak passwords can be broken using wordlists and GPU/CPU power. This project helped me see why strong passwords are important in cybersecurity.

Tools Used

  • Hashcat
  • Linux Terminal (Kali Linux)
  • rockyou.txt wordlist

Steps and Commands

1. Create a test password hash:

echo -n "Manjeet123" | md5sum

2. Save the hash inside a file:

echo "5c768a66e1d540b7350a5f0c1f7bcc1f" > hash.txt

3. Run Hashcat to crack the hash:

hashcat -m 0 hash.txt /usr/share/wordlists/rockyou.txt

4. Show the cracked password:

hashcat -m 0 hash.txt --show

What I Learned

I learned how hashes work, how password cracking tools operate, and why companies use strong hashing algorithms. This project helped me understand the attacker's side and why security teams must enforce strong password rules.
