Nmap Network Scanning

Cybersecurity Project by Manjeet Singh

Project Summary

In this project, I used Nmap to scan networks, check for open ports, and identify running services. This is one of the basic skills needed in cybersecurity because it helps you understand what systems are exposed on a network.

Tools Used

  • Nmap
  • Linux Terminal (Kali or Ubuntu)

Commands I Used

  • nmap -sV 192.168.1.1 – scans a device and shows services
  • nmap -sC -sV 192.168.1.1 – runs default scripts
  • nmap -p- 192.168.1.1 – scans all 65,535 ports
  • nmap -A 192.168.1.1 – aggressive scan (OS detection, services, scripts)

What I Learned

I learned how to scan a network safely, how to check for open ports, and how to identify the services running on different machines. This helps in both ethical hacking and defensive security work because it shows what an attacker might see.
