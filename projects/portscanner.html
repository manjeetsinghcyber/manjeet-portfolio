Python Port Scanner Tool

Cybersecurity Project by Manjeet Singh

Project Summary

In this project, I created a simple port scanner in Python. It scans a target IP address or website to check which ports are open. This project helped me understand how network scanning works and how tools like Nmap are built.

Tools Used

  • Python 3
  • socket library
  • Kali Linux / Ubuntu / Windows

Python Code


import socket

target = input("Enter the IP address to scan: ")

print(f"\nScanning target: {target}\n")

for port in range(1, 1025):
    s = socket.socket(socket.AF_INET, socket.SOCK_STREAM)
    s.settimeout(0.5)
    result = s.connect_ex((target, port))

    if result == 0:
        print(f"Port {port} is OPEN")
    s.close()

print("\nScan complete.")

Steps Performed

1. Wrote the Python script using socket.

2. Ran the scanner on localhost.

python3 portscanner.py

3. Tested different ports (FTP, SSH, HTTP, HTTPS).

4. Confirmed results using Nmap to compare accuracy.
