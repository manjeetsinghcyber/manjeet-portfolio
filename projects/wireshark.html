Wireshark Packet Capture & Analysis

Cybersecurity Project by Manjeet Singh

Project Summary

In this project, I used Wireshark to capture and analyze network traffic. I learned how different protocols work, how network packets look, and how analysts detect suspicious behavior on a network.

Tools Used

  • Wireshark
  • Kali Linux / Windows
  • Packet filters

Steps Performed

1. Started a packet capture:

Open Wireshark → Select network interface → Start Capture

2. Applied filters to analyze specific traffic:

  • http – shows HTTP traffic
  • dns – shows DNS queries
  • icmp – shows ping packets
  • tcp.port == 80 – HTTP on port 80

3. Analyzed packet details:

Frame → Ethernet → IP → TCP

4. Identified source/destination IPs and protocols.

5. Saved the capture as a .pcap file.

What I Learned

I learned how network packets are structured, how protocols communicate, how to filter traffic, and how analysts detect suspicious connections. This project helped me understand real-world blue team work.
